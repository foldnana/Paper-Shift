# 《纸上打工》完整设计交付包

这是《纸上打工》的当前完整设计交付文件夹，供后续策划、UI、开发继续设计和实现。

## 文件说明

| 文件 | 用途 |
|---|---|
| `01_完整游戏设计稿.md` | 游戏定位、核心循环、时代/职业/标签/退休等完整设计 |
| `02_UI原型与页面流程.md` | 当前 UI 原型的页面拆解、交互流程、视觉规范 |
| `03_内容与数值模板.md` | 职业、标签、事件、退休原因、预算效果等内容模板 |
| `纸上打工_UI原型.html` | 可直接打开的网页 UI 原型，仿《纸上谈亲》风格 |

## 如何查看 UI 原型

直接双击打开：

`纸上打工_UI原型.html`

如果浏览器限制本地文件，也可以在该目录启动本地服务器：

```bash
python -m http.server 8765
```

然后访问：

`http://127.0.0.1:8765/纸上打工_UI原型.html`

## 当前设计目标

《纸上打工》不是传统经营游戏，而是一个类《纸上谈亲》的休闲文字 Roguelite：

- 用“求职、打工、预算、退休、下一代”替代“相亲、相处、结婚、生子、下一代”。
- 用职业、时代、简历包装、工资分配和退休原因制造轻量但有梗的故事。
- UI 保持《纸上谈亲》的纸片卡片、蓝底、票据标签、弹窗选择风格。
