# 《纸上打工》完整游戏设计稿

## 一、游戏定位

《纸上打工》是一款休闲文字 Roguelite 小游戏。玩家扮演不同年代的普通劳动者，从创建角色开始，经历简历包装、求职面试、入职打工、工资分配、恋爱结婚、生育教育、退休退场，并把部分财富、标签和负担留给下一代。

核心体验不是复杂经营，而是：

- 一眼看懂角色命运。
- 一次选择改变人生走向。
- 每一代都有荒诞又合理的打工故事。
- 失败不是结束，而是下一代开局素材。

## 二、参考与替换关系

本作核心结构参考《纸上谈亲》：

| 《纸上谈亲》 | 《纸上打工》 |
|---|---|
| 创建相亲对象 | 创建劳动者 |
| 选择标签 | 选择人生/能力/负担标签 |
| 隐藏信息 | 编辑简历，显示/隐藏/夸大/无中生有 |
| 相亲对象 | 面试岗位 |
| 继续相处 | 面试第几轮 |
| 确认关系 | 入职 |
| 继续相处后的事件 | 打工一个月后的事件 |
| 恋爱/结婚/生子 | 通过预算系统触发恋爱、结婚、生子 |
| 代际传承 | 退休退场，开启下一代 |

## 三、核心循环

### 1. 创建劳动者

玩家生成一个角色，包含：

- 性别
- 时代
- 年龄
- 体魄
- 识字
- 逻辑
- 社交
- 学历
- 家境
- 初始优势
- 初始资产

创建阶段不出现职业和工资，因为角色还没有开始工作。

### 2. 选择标签

玩家选择 3 个标签。标签决定后续求职、打工和退休结果。

示例：

- `会算账`：财务和交易事件更有利。
- `能吃苦`：基础体力职业更稳。
- `同乡介绍`：更容易获得面试机会，但欠人情。
- `祖传手艺`：工匠路线成长快，可影响下一代。
- `AI先行者`：未来职业和技术岗位更容易成功。
- `社恐`：客户/面试类事件不利，但独立工作压力低。

### 3. 编辑简历

简历页是本作的核心题材特色。

玩家可以对自己的信息选择：

- `显示`：真实展示，稳定但不冒险。
- `隐藏`：减少短板暴露，但后续可能被发现。
- `夸大`：提高面试概率，但增加入职后暴雷风险。
- `无中生有`：大幅提高短期机会，但风险最高。

简历还包含“求职意向”，决定系统优先刷出的工作类型。

示例求职意向：

- AI/技术岗
- 远程优先
- 月薪过万
- 稳定体制
- 手艺学徒
- 高薪高危
- 自由职业

### 4. 求职面试

求职页采用上下双卡结构：

- 上方：玩家求职者卡。
- 下方：当前面试岗位卡。

底部按钮：

- `面试第2轮`：推进面试流程。
- `再投一家`：刷新新的岗位机会。
- `就这份了`：面试通过或达到阈值后入职。

岗位会根据求职意向、时代、标签和属性生成。

### 5. 面试失败

失败时用灰色横幅覆盖下方岗位卡，类似《纸上谈亲》关系断掉的反馈。

示例：

```text
对方认为你缺少岗位经验，不再和你联系。
```

失败会产生：

- 压力上升。
- 时间推进。
- 可能获得标签，如 `面试阴影`、`被学历卡住`、`简历暴雷`。

### 6. 入职打工

入职后进入打工页：

- 上方：玩家当前状态。
- 下方：已入职岗位信息。

按钮：

- `再干一月`：推进一个月。
- `分配`：进入工资预算页。

每次打工结算：

- 工资到账
- 饮食支出
- 住房支出
- 恋爱/社交支出
- 教育/学习支出
- 存款变化
- 随机事件
- 压力变化

### 7. 新闻弹窗

职业新闻不做独立页面，只作为当前界面上的弹窗事件出现。

示例：

```text
新闻：
新职业「人机团队协调员」出现了。

选项：
1. 投相关岗位
2. 先干当前工作
3. 攒钱观望
```

选项最多 3 个。

### 8. 工资分配

预算页是人生系统的入口。

玩家按比例分配工资：

- 饮食
- 住房
- 恋爱
- 教育
- 存款

预算影响：

| 项目 | 影响 |
|---|---|
| 饮食 | 健康恢复，压力缓解 |
| 住房 | 结婚稳定度，家庭满意度 |
| 恋爱 | 恋爱、结婚、生子事件概率 |
| 教育 | 后代学历、技能、初始标签 |
| 存款 | 退休财产和抗风险能力 |

### 9. 退休结算

一代结束不叫传承，改为“退休结算”。

退休/退场原因包括：

- 到龄退休
- 意外死亡
- 压力过大进精神病院
- 工伤退场
- 裁员后长期失业
- 破产返乡
- 提前财富自由
- 行业消失被迫退出

退休页展示：

- 退场原因
- 工作年限
- 最终职业
- 留下存款
- 精神状态
- 后代列表
- 退休财产分配
- 开启下一代

## 四、时代系统

时代不是皮肤，而是决定职业池、风险和机会。

| 时代 | 核心内容 |
|---|---|
| 古代农耕 | 佃农、工匠、书吏、兵役、灾荒、地主 |
| 近代工业 | 工厂、铁路、学徒、工伤、城市迁移 |
| 现代城市 | 平台、公司、房租、简历、裁员、自由职业 |
| 近未来智能 | AI 替代、AI 协作、远程办公、新职业 |
| 远未来星际 | 轨道矿工、生态舱、星舰护卫、氧气债务 |
| 后工作时代 | 基本保障、意义劳动、手艺复兴、精神空虚 |

## 五、职业系统

职业不再是时代专属，而是“职业原型 + 时代形态”。

示例：农业者

| 时代 | 职业形态 |
|---|---|
| 古代农耕 | 佃农、自耕农 |
| 近代工业 | 小农、进城农民工 |
| 现代城市 | 农场工、农业主播 |
| 近未来智能 | 智慧农业操作员 |
| 远未来星际 | 生态舱种植员 |
| 后工作时代 | 生态园守护者 |

## 六、胜负与长期目标

本作没有单一胜利，核心是人生小传和家族推进。

一代目标可以是：

- 活到退休
- 攒下财产
- 获得稀有职业
- 避开精神崩溃
- 培养优质后代
- 把家族带入新时代

失败也要有趣：

- 压力过大进精神病院
- 简历造假被全行业拉黑
- 工伤退场但获得赔偿
- 被 AI 替代后开了手作小店
- 到龄退休但孩子不愿接班
